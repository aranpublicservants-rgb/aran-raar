Anti-Racism Ambassadors Network / Réseau des ambassadeurs anti-racisme
This repository contains a standalone GitHub Pages file for the Anti-Racism Ambassadors Network / Réseau des ambassadeurs anti-racisme (ARAN-RAAR).
Repository description
Anti-Racism Ambassadors Network: supporting anti-racism, knowledge sharing, collaboration and collective action across public service communities. Réseau des ambassadeurs anti-racisme : appuyer l’antiracisme, le partage des connaissances, la collaboration et l’action collective.

Purpose
This page explains what ARAN-RAAR is, what it does, how public servants can connect, and where to find related resources. It is written in plain language and avoids naming individual network leads.
What the page includes
Network overview
Purpose and role
Key activities
Ways to get involved
Direct email link to the network
Links to related GCcollab, GCcollab Wiki, GCXchange, Community Agreement Form, Trello resource library, podcast and LinkedIn pages
English and French language toggle
Accessibility controls for large print, increased spacing, high contrast and reduced motion
Embedded network logo
Accessibility approach
The page is designed to support Government of Canada web accessibility expectations and WCAG-oriented practices. It includes:
Semantic HTML structure
Skip link
Keyboard-accessible buttons and links
Visible focus indicators
Plain language content
Responsive layout for mobile and desktop screens
No reliance on colour alone
User-controlled display options
Final accessibility compliance should still be checked with manual keyboard testing, screen reader testing, colour contrast testing and link validation before publication.
Suggested GitHub Pages setup
Create a repository named:
aran-raar
Upload the page file.
Rename aran-raar.html to index.html if it should open automatically as the homepage.
Add this README.md file to the repository.
In GitHub, go to Settings > Pages.
Under Build and deployment, select the branch that contains the page, usually main.
Save the Pages settings and wait for GitHub to publish the site.
